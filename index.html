무냐가 해주는 것들

복잡한 설정 없이 슬래시 명령어 하나로 모든 걸 패널로 관리해요

🛡️

서버 보안

도배·초대링크·전체멘션·금지어 자동 감지. 화이트리스트, 블랙리스트, 경고 시스템으로 서버를 안전하게 지켜요.

📊

활동량 통계

채팅·음성 채널 활동을 자동 집계해요. 멤버별 기여율, 채널 통계, TOP 15 랭킹을 한눈에 확인할 수 있어요.

⚙️

서버 관리

경고 추가·확인·초기화, 타임아웃, 킥, 메시지 정리, 공지 전송. 버튼 하나로 모든 관리 작업 완료.

🎭

반응 역할

이모지를 클릭하면 자동으로 역할이 지급돼요. 메시지 생성부터 이모지 등록까지 명령어 하나로 끝.

🎫

티켓 시스템

버튼 클릭 한 번으로 개인 채널 자동 생성. 문의·신고를 체계적으로 관리하고 로그도 자동으로 남아요.

🎉

이벤트 & 끝말잇기

응모 이벤트 생성, 추첨, 당첨 확률 표시. 끝말잇기로 서버 분위기도 UP! 봇이 자동으로 다음 단어 이어받아요.
